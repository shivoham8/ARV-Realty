# 🏢 ARV Realty Website

A modern, responsive real estate website designed to showcase residential and commercial projects with an engaging user experience.

---

## 🚀 Features

- 🎥 **Video-based Hero Section** for a premium first impression  
- 🧭 **Responsive Navigation Bar** with toggle menu  
- 🏗️ **Projects Showcase** with interactive image galleries  
- 🖼️ **Lightbox Gallery** for viewing project images without leaving the page  
- 🎠 **Swiper.js Sliders** for smooth transitions  
- 👨‍💼 **Leadership Section** highlighting key members  
- 📍 **Google Maps Integration** for location  
- 📩 **Contact Form UI**  
- 📱 Fully **Responsive Design** for mobile, tablet, and desktop  

---

## 🛠️ Tech Stack

- **HTML5**
- **CSS3**
- **JavaScript (Vanilla JS)**
- **Swiper.js** (for sliders)
- **Lightbox2** (for image galleries)
- **Font Awesome** (for icons)

---

---

## ⚙️ How It Works

### 🔹 Project Gallery
- Clicking the **"+" icon** loads project-specific images dynamically using JavaScript.
- Images are grouped using `data-lightbox` to create separate galleries for each project.
- Lightbox2 handles the modal view and navigation.

### 🔹 Sliders
- **Swiper.js** is used for:
  - Homepage hero slider  
  - Leadership section carousel
